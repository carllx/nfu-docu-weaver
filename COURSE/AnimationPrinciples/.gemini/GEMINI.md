# Gemini Project Memory

## Project Overview

这是一个关于动画原理的课程项目，主要使用 Obsidian 来组织和显示 Markdown 文件。

**核心内容:**
*   **主题:** 以 12 条经典动画原理 (12 Animation Principles) 为基础。
*   **实践工具:** Blender。
*   **技术范围:** 包含 3D 动画、2D Grease Pencil 动画，以及物理引擎动画。
*   **资料收集:** 教程、博客文章、书目素材。
*   **课程产出:** 课程大纲、18 周的 Marpit 格式 PPT 课件。
