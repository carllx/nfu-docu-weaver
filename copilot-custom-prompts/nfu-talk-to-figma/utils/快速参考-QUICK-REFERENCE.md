| 项目 | 推荐值 |
|------|--------|
| 面数（单个） | < 50,000 |
| 面数（场景） | < 300,000 |
| 材质数量 | ≤ 10 个 |

| 文件大小 | < 100MB |

### 纹理规格
| 类型 | 格式 | 尺寸 |
|------|------|------|
| Diffuse/Albedo | JPG/PNG | 2048² |
| Normal | PNG | 2048² |
| Roughness | JPG | 1024² |
| Metallic | JPG | 1024² |
| AO | JPG | 1024² |

**规则**: 必须是 2 的幂次方（512, 1024, 2048, 4096）

## ⚙️ FBX 导出设置（关键）

```
File → Export All... → FBX export

✅ 必须勾选：
   Embed Media ⭐⭐⭐（最重要！）
   Smoothing Groups
   
✅ 推荐设置：
   Type: Binary（二进制）
   Version: FBX 2020
   
✅ 如果有动画：
   Animation
   Bake Animation
```

## 🎨 PBR 材质参数

### 基础参数
| 参数 | 说明 | 常用值 |
|------|------|--------|
| **Base Color** | 基础颜色 | 纹理或 RGB |
| **Metallic** | 金属度 | 0（非金属）/ 1（金属） |
| **Roughness** | 粗糙度 | 0.2-0.8 |
| **Normal** | 法线贴图 | 强度 0.5-2.0 |
| **AO** | 环境光遮蔽 | 强度 0.5-1.5 |

### 材质类型快速设置
```
木头：
Metallic: 0
Roughness: 0.6-0.8

塑料：
Metallic: 0
Roughness: 0.3-0.5

金属（铁）：
Metallic: 1
Roughness: 0.4-0.6

金属（金）：
Metallic: 1
Roughness: 0.2-0.3
Base Color: RGB(255, 215, 0)

玻璃：
Metallic: 0
Roughness: 0.0-0.1
Opacity: 0.1-0.3
```

## 💡 三点光照快速设置

```
主光（Key Light）：
├─ Type: Directional
├─ Intensity: 1.5-2.0
├─ Color: 略偏暖（#FFF5E1）
└─ Position: 前上方 45°

补光（Fill Light）：
├─ Intensity: 0.5-1.0
├─ Color: 略偏冷（#E1F5FF）
└─ Position: 侧面或对面

边缘光（Rim Light）：
├─ Intensity: 0.3-0.8
├─ Color: 纯白或略偏蓝
└─ Position: 背后上方
```

## 🔧 常见问题速查

### 纹理丢失
```
原因: FBX 未勾选 "Embed Media"
解决: 重新导出，勾选该选项
```

### 模型全黑
```
原因: Metallic 过高或光照不足
解决: 
1. 降低 Metallic 到 0
2. 增加 Roughness 到 0.5
3. 增加环境光 Exposure
```

### 模型全白
```
原因: Roughness 过低或缺少纹理
解决:
1. 增加 Roughness 到 0.5
2. 检查纹理是否加载
3. 调整 Base Color
```

### 文件过大
```
解决方法（按优先级）:
1. 压缩纹理（4K → 2K）
2. JPG 替代 PNG
3. 减少模型面数
4. 使用 Binary FBX
5. 合并材质
```

### UV 错误
```
检查:
1. UV Editor 中查看展开
2. 检查是否有重叠
3. 确认在 0-1 范围内
4. 检查是否有反向 UV
```

## 🚀 优化技巧

### 减少面数
```
Mesh → Reduce
Reduction %: 50-70%
保持轮廓的同时优化
```

### 清理模型
```
Mesh → Cleanup
勾选:
- Remove zero geometry
- Remove zero edge length
- Remove non-manifold geometry
```

### 检查法线
```
Display → Polygons → Face Normals
确保所有法线朝外
反面修复: Mesh Display → Reverse
```

### 优化纹理
```
1. 降低分辨率（4K → 2K）
2. 使用 JPG（质量 85%）
3. 合并相似纹理
4. 移除未使用的纹理
```
