## 🎯 核心代码模式

### 创建按钮框架
```javascript
create_frame(
  x, y, width, height,
  name: "Button/Type/Size/State/Icon",
  fillColor: {r, g, b},
  layoutMode: "HORIZONTAL",
  primaryAxisAlignItems: "CENTER",
  counterAxisAlignItems: "CENTER",
  padding: {...},
  itemSpacing: 8
)
```

### 添加文本
```javascript
create_text(
  parentId: button_id,
  text: "Button Label",
  fontSize: 16,
  fontWeight: 600,
  fontColor: {r, g, b}
)
```

### 添加图标占位符
```javascript
create_rectangle(
  parentId: button_id,
  width: 20, height: 20,
  fillColor: {r: 1, g: 1, b: 1, a: 0.8}
)
```

### 设置圆角
```javascript
set_corner_radius(nodeId, radius: 8)
```

### 最后选中所有按钮
```javascript
set_selections(["30:4", "30:8", ...])
```
