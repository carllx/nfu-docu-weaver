### Reference Pyramids

In the Reference Pyramids method, ‚ÄúClassic Literature‚Äù usually refers to the earliest literature that proposes theories or frameworks in a research field, while ‚Äúcore literature‚Äù refers to literature that constructively develops or applies those classic works.

When collecting literature on the topic of ‚ÄúDetecting Memory‚Äù, we first identified the top 4 most frequently co-cited papers as the ‚ÄúClassic Literature.‚Äù We then counted the number of papers citing each classic paper, and ranked them by citation count. We selected papers with over 45 citations as the ‚Äúcore literature‚Äù derived from each classic work. Below, we outline the classic and corresponding core literature in a table format.

The table headers are the Classic Literature. ‚ÄúTI‚Äù indicates the paper title, ‚ÄúPY‚Äù the publication year, and ‚ÄúTC‚Äù the number of citations for each core paper.
